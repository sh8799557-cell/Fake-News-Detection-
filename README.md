

An outcome-based comparative research framework analyzing sequential deep learning networks against context-vectorized classification matrices for automated fake news identification.

* `/data`: Preprocessed and curated binary-labeled master datasets.
* `/notebooks`: Google Colab source execution files for Bi-LSTM and TF-IDF pipelines.
* `/outputs`: Evaluated visual graphics including system Confusion Matrices.

Our experimental models evaluated on the benchmark ISOT Fake News Dataset achieved the following optimization parameters:

| Architectural Setup | Test Precision | Test Recall | Overall Accuracy |
| :--- | :--- | :--- | :--- |
| **Bidirectional LSTM** | 0.92 | 0.93 | **92.50%** |
| **TF-IDF + Linear Classifier** | 0.98 | 0.97 | **97.64%** |

* TensorFlow / Keras
